## Niku Row
Style property builder for building "Row".

## Example usage:
As Widget
```dart
NikuRow([
  Text("1"),
  Text("2"),
])
  ..mainStart();
```

As Property
```dart
Row(
  children: [
    Text("1"),
    Text("2"),
  ]
)
  .asNiku()
  ..mainStart();
```

## Availability 
To use this widget, you can import from the following:

```dart
// All Widget
import 'package:niku/niku.dart';

// Extension
import 'package:niku/extension/widget.dart';

// With relatated widget
import 'package:niku/widget/axis.dart';

// Just widget
import 'package:niku/widget/row.dart';
```

## niku
Switch to use parent property builder.

Example Usage:
```dart
NikuRow()
  .niku();
```

Equivalent to
```dart
Niku(
  Row(),
);
```

## mainAxisAlignment
Set main axis alignment.

Example Usage:
```dart
NikuRow()
  ..mainAxisAlignment(MainAxisAlignment.center);
```

Equivalent to
```dart
Row(
  mainAxisAlignment: MainAxisAlignment.center,
);
```
