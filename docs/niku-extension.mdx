# Niku Extension
This extension add `.niku()`, parent property builder extension to all `Widget`.

## Example usage:
```
Container()
  .asNiku()
  ..bg(Colors.blue)
  ..p(21)
```

## Availability 
```dart
// All Widget
import 'package:niku/niku.dart';

// Just extension
import 'package:niku/extension/niku.dart';
```
