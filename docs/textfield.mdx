## Niku TextField
Style property builder for building "TextField" and "TextFormField".

## Example usage:
As Widget
```dart
NikuTextField("Username")
  ..outlined(
    color: Colors.blue,
  );
```

As Property
```dart
TextField("Username")
  .asNiku()
  ..outlined(
    color: Colors.blue,
  );
```

## Availability: 
To use this widget, you can import from the following:

```dart
// All Widget
import 'package:niku/niku.dart';

// Extension
import 'package:niku/extension/widget.dart';

// Just widget
import 'package:niku/widget/text.dart';
```

## niku
Switch to use parent property builder.

Example Usage:
```dart
NikuTextField("Username")
  .controller(controller)
  .outlined(
    color: Colors.blue,
    width: 2,
  )
  .niku()
  .px(8);
```

Equivalent to
```dart
Niku(
  Padding(
    padding: EdgeInsets.symmetric(
      horizontal: 8,
    ),
    NikuTextField(
      controller: controller,
      decoration: InputDecoration(
        labelText: "Username",
        border: OutlineInputBorder(
          borderSide: BorderSide(
            color: color,
            width: width,
          ),
        ),
        enableBorder: OutlineInputBorder(
          borderSide: BorderSide(
            color: color,
            width: width,
          ),
        ),
        disabledorder: OutlineInputBorder(
          borderSide: BorderSide(
            color: color,
            width: width,
          ),
        ),
        focusedBorder: OutlineInputBorder(
          borderSide: BorderSide(
            color: color,
            width: width,
          ),
        ),
        errorBorder: OutlineInputBorder(
          borderSide: BorderSide(
            color: color,
            width: width,
          ),
        ),
      ),
    ),
  ),
);
```

## apply
Apply predefined style properties to widget.

Example Usage:
```dart
final title = NikuTextField()
  ..color(Colors.red)
  ..fontSize(21);

NikuTextField()
  ..apply(title);
```

Equivalent to
```dart
Text()
  ..color(Colors.red)
  ..fontSize(21);
```
