# Icon
A representation of [Icon](https://material.io/design/iconography/system-icons.html).

`NikuIcon` accepts all the props as same as [Icon](https://api.flutter.dev/flutter/widgets/Icon-class.html).

```dart
// Using namespace
n.Icon(Icons.home)

// Using Niku Prefix
NikuIcon(Icons.home)
```

## Example Usage
```dart
n.Icon(Icons.home)
    ..color = Colors.blue
    ..size = 24;
```
