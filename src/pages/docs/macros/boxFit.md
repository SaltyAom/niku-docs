Reusable shortcuts abstract class used in:
- [Image](/docs/widgets/image)


To use this macro, simply import from one of the following:
```dart
import 'package:niku/niku.dart';
import 'package:niku/macros/macros.dart';
import 'package:niku/macros/BoxFitMacro.dart';
```
## BoxFitMacro

### contain `shortcut`

Equivalent to:
```dart
fit = BoxFit.contain
```

### cover `shortcut`

Equivalent to:
```dart
fit = BoxFit.cover
```

### fill `shortcut`

Equivalent to:
```dart
fit = BoxFit.fill
```

### fitHeight `shortcut`

Equivalent to:
```dart
fit = BoxFit.fitHeight
```

### fitWidth `shortcut`

Equivalent to:
```dart
fit = BoxFit.fitWidth
```

### noFit `shortcut`

Equivalent to:
```dart
fit = BoxFit.none
```

### scaleDown `shortcut`

Equivalent to:
```dart
fit = BoxFit.scaleDown
```
